ESBoot provides out-of-the-box unit testing through the [plugin-vitest](../plugins/plugin-vitest) plugin.

That gives projects a default path for:

- component tests
- interaction tests
- async state tests
- a lightweight local test workflow without assembling the tooling stack manually

See the plugin page for installation and usage details.
