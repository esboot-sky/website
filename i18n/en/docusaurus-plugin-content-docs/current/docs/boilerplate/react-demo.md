This is a minimal template intended for quick demos.

## File Structure

```bash
.
├── config
├── package.json
├── src
│   ├── app.tsx
│   ├── index.css
│   └── index.entry.tsx
└── tsconfig.json
```

It keeps the project small while still preserving standard ESBoot capabilities. The default bundler is Vite.

## Create

See [Create a Project](../getting-started#option-2-create-from-built-in-templates) and select the `React-DEMO-APP` template.
