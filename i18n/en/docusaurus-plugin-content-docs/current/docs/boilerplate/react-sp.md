This is a minimal single-platform template for quick starts and demos.

## File Structure

```bash
.
├── config
├── package.json
├── src
│   ├── app.tsx
│   ├── index.css
│   └── index.entry.tsx
└── tsconfig.json
```

It stays compact while keeping standard ESBoot features. The default template uses Vite.

## Create

See [Create a Project](../getting-started#option-2-create-from-built-in-templates) and choose the built-in React demo app template.
