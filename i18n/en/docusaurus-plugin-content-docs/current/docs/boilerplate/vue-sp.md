This is a minimal single-platform Vue template for demos and small starters.

## File Structure

```bash
.
├── config
├── package.json
├── src
│   ├── app.tsx
│   ├── index.css
│   └── index.entry.tsx
└── tsconfig.json
```

It keeps the footprint small while preserving standard ESBoot capabilities. The default template uses Vite.

## Create

See [Create a Project](../getting-started#option-2-create-from-built-in-templates) and choose the built-in Vue template.
