# 说明

为了方便写demo，生成的一个最小模版。

## 文件目录

```bash
.
├── config
├── package.json
├── src
│   ├── app.tsx
│   ├── index.css
│   └── index.entry.tsx
└── tsconfig.json
```

最小化体积，但是拥有完整的esboot功能，默认使用Vite模版。

## 创建

参考[创建项目](../../getting-started#方式一从内置模版创建)。

选择模版`React-DEMO-APP`。
